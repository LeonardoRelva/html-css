Principais formatações

Negrito / Destaque

Nesta frase terá um termo em negrito usando a tag <B> (sem semântica).

Nesta frase, temos um termo em destaque usando a tag <STRONG> (semântica).

Itálico / Ênfase

Neste perído temos um termo em itálico usando a tag <I> (sem semântica).

Neste período, temos um termo em ênfase usando a tag <EM> (semântica).

OBSERVAÇÃO!

Segurando Ctrl+Shift+P podemos envelopar (marca a parte que deseja colocar a tag e aperta o comando).

<strong> = NEGRITO COM SEMÂNTICA (DESTAQUE).

<em> = ITÁLICO COM SEMÂNTICA (ÊNFASE).

Texto marcado

Se quiser colocar um texto marcado basta colocar a tag <MARK>.

E no outro parágrafo, temos um texto marcado no final.

OBSERVAÇÃO! = Se quiser deixar todas as marcas com uma mesma cor, basta colocar style na parte de "head", depois colocar a tag "MARK", depois colocar a BACKGROUND-COLOR e escolher a cor da marcação.

Texto GRANDE e Texto pequeno

Estamos criando um texto grande e um texto pequeno no mesmo parágrafo. (tag BIG E tag SMALL)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. (tag DEL)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfasee indicar que ele foi adicionado depois. (tag INS)

Texto Sobrescrito

Para indicar coisas do tipo x20+7

Texto Subscrito

Para indicar coisas do tipo H2O

TAGS APRENDIDAS NESSE EXERCÍCIO

tag STRONG para deixar em negrito

tag EM para deixar em itálico

tag MARK para deixar o texto marcado

tag BIG para deixar a letra grande (não semântica)

tag SMALL para deixar o texto pequeno

tag DEl para deixar o texto deletado

tag INS para deixar o texto sublinhado

tag SUP para deixar a letra em cima das outras (exemplo = x8)

tag SUB para deixar a letra em baixo das outras (exemplo = H20)
